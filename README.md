# PlanLlama AI Experiment

An attempt to write an AI Agent in PlanLlama

## Tech Stack

- React 19.2.0
- Vite 7.2.2
- Modern JavaScript (ES modules)

## Features

- Multi-line text input for user queries
- Submit button to process input
- Results display area showing submission history with timestamps
- Modern dark/light theme support
- Ready for AI agent integration

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Lint code
npm run lint
```

## Project Structure

- `src/App.jsx` - Main application component with form and results display
- `src/main.jsx` - Application entry point
- `src/App.css` - Component-specific styles
- `src/index.css` - Global styles
